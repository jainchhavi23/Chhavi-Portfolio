## 📥 Installation

Clone the repository

```bash
git clone https://github.com/jainchhavi23/Chhavi-Portfolio.git
```

Move into the project directory

```bash
cd Chhavi-Portfolio
```

Create a virtual environment

```bash
python -m venv .venv
```

Activate the virtual environment

### Windows

```bash
.venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
